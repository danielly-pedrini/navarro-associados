# ⚖️ Navarro & Associados — Site Institucional


## 📋 Sobre o Projeto

Site institucional moderno e responsivo desenvolvido para o escritório de advocacia **Navarro & Associados**. O projeto foi construído com foco em elegância visual, performance e experiência do usuário — refletindo os valores do escritório: seriedade, confiança e excelência jurídica.

O design adota uma identidade sofisticada em tons escuros com acentos dourados, tipografia refinada e animações sutis que transmitem autoridade e modernidade.

## ✨ Funcionalidades

- **Hero Section** com chamada para ação, badges de credibilidade e foto da sócia fundadora
- **Barra de estatísticas** animada com números de impacto (15+ anos, 2.400+ clientes atendidos)
- **Seção Sobre** com apresentação do escritório e valores institucionais
- **Áreas de Atuação** em grid com cards interativos (Família, Trabalhista, Criminal, Empresarial, Previdenciário e Cível)
- **Diferenciais** destacados com layout numerado
- **Equipe** com cards fotográficos e overlay em hover
- **CTA integrado ao WhatsApp** com mensagem de urgência e sigilo
- **Seção de Credibilidade** reforçando a promessa da marca
- **Rodapé completo** com links, áreas e identidade visual
- **Navbar fixa** com comportamento de scroll e blur progressivo
- **Scroll suave** e animações de entrada com Intersection Observer

## 🎨 Design & Identidade Visual

| Atributo       | Detalhe                                      |
|----------------|----------------------------------------------|
| **Paleta**     | Dark `#0E0E0E` + Gold `#B89A5A`              |
| **Tipografia** | Cormorant Garamond (serif) + Jost (sans)     |
| **Tema**       | Dark luxury — sofisticado e autoritário      |
| **Animações**  | Fade-up no scroll via IntersectionObserver   |
| **Layout**     | CSS Grid + Flexbox, sem frameworks externos  |

---

## 📱 Responsividade

O layout é totalmente responsivo, adaptando-se a qualquer dispositivo:

| Breakpoint   | Comportamento                                        |
|--------------|------------------------------------------------------|
| **Desktop**  | Layout em grid de 2–4 colunas, hero com imagem       |
| **Notebook** | Ajuste proporcional de fontes e espaçamentos         |
| **Tablet**   | Grid reduzido para 2 colunas, navegação simplificada |
| **Mobile**   | Layout em coluna única, nav sem links (mobile-first) |

## 🚀 Como Executar

Nenhuma dependência ou build necessário. Basta abrir o arquivo diretamente no navegador:

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/navarro-associados.git

# Acesse a pasta
cd navarro-associados

# Abra no navegador
open index.html
```

> As fontes são carregadas via Google Fonts. É necessário conexão com a internet para exibi-las corretamente.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** semântico
- **CSS3** puro — variáveis, grid, flexbox, animações e pseudo-elementos
- **JavaScript** vanilla — IntersectionObserver, scroll handler, smooth scroll
- **Google Fonts** — Cormorant Garamond & Jost
- **Zero dependências externas** — sem frameworks, sem bibliotecas
